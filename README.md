> ## How to use

```
  # Clone repository:
  git clone https://github.com/Mist3rBru/base-converter-api.git;
  cd base-converter-api;

  # Install dependencies:
  npm install;

  # Run server:
  npm run dev;

  # Access http://localhost:4000
```
